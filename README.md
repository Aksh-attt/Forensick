# Forensick

A 2D psychological horror survival game built in Unity.

## Gameplay
Survive as long as possible while hallucinations close in around you. 
Your torch slowly dims as darkness consumes the world.

## Features
- Torch-based visibility system
- Progressive darkness mechanic
- Wave-based enemy spawning
- Whip combat system
- Enemy health bars and damage indicators

## Tech Stack
- Unity 2D (URP)
- C#
- Spring Boot Leaderboard API (coming soon)
